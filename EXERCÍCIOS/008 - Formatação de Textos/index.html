Principais formatações

Negrito / Destaque

Existem duas maneiras de criar uma palavra em negrito a "semântica" e a "não semântica".

Ex.1: Nesta frase temos um Termo em Negrito usando a tag <b> (não semântica).

Ex.2: Nesta frase, temos um Termo em Destaque usando a tag <strong> (semântica).


Itálico / Ênfase

Ex.1: Nesta frase temos um Termo em Itálico usando a tag <i> (não semântica).

Ex.2: Nesta frase temos um Termo em ênfase usando a tag <em> (semântica).


Texto marcado

Podemos criar também um texto marcado usando a teg <mark>.


Texto grande e texto pequeno

Estamos criando um texto grande (<big>) e um texto pequeno (<small>) neste parágrafo.


Texto deletado

Podemos marar um texto como excluído para indicar que ele deve ser lido, mas não considerado.


Texto inserido

Podemos marcar um texto como inserido para ênfase e indicar que ele foi adicionado depois.

Existe o texto sublinhado com a tag <u> (não semântica).


Texto sobrescrito

Para inserir textos do tipo x20+3 ou y2+x.


Texto subscrito

Para inserir textos com H2PO4 ou H2O.
